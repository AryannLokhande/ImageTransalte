📘 About Image Translate Web App

This is an AI-powered Image Translate Multi-Language Web Application. It allows users to upload images or enter text and translate it into different languages.

🚀 Features

📸 Image Text Detection (OCR)
🌐 Multi-Language Translation
📝 Text Input Translation
📜 Translation History
⚡ Fast Processing

🎯 Purpose

This application helps users understand foreign language text from images quickly. It uses OCR technology to detect text and then translates it into the selected language.
